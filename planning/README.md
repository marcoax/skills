# planning/

Planning, spec, and triage workflows.

> When you add, rename, or remove a skill in this folder, update the top-level `README.md`.

| Skill | Description |
|-------|-------------|
| [planning-with-files](planning-with-files/SKILL.md) | File-based planning (`task_plan.md`, `findings.md`, `progress.md`) for complex multi-step tasks. |
| [prd-to-plan](prd-to-plan/SKILL.md) | Turns a PRD into a multi-phase implementation plan using tracer-bullet vertical slices. |
| [task-spec-creator](task-spec-creator/SKILL.md) | Generates a structured `TASK_SPEC.md` for a single implementation task via developer interview. |
| [triage-issue](triage-issue/SKILL.md) | Triages a bug, finds root cause, and creates a GitHub issue with a TDD-based fix plan. |

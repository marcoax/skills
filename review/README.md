# review/

Code review variants.

> When you add, rename, or remove a skill in this folder, update the top-level `README.md`.

| Skill | Description |
|-------|-------------|
| [optimistic-code-review](optimistic-code-review/SKILL.md) | Constructive code review — surfaces issues with WHY + HOW, acknowledges strengths, proposes fixes before applying. |
| [pessimistic-code-review](pessimistic-code-review/SKILL.md) | Adversarial review using Independent Adversarial Verification (IAV); returns evidence-based PASS/FAIL. |
| [react-review](react-review/SKILL.md) | React 18/19 specific review: hooks, state, rendering, data fetching, component APIs. |
| [goal-spec-review](goal-spec-review/SKILL.md) | Pre-flight audit of a spec — verifies clear objectives and explicit completion criteria before coding begins. |

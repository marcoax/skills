# skill-management/

Meta-skills to author and tune other skills.

> When you add, rename, or remove a skill in this folder, update the top-level `README.md`.

| Skill | Description |
|-------|-------------|
| [autoresearch](autoresearch/SKILL.md) | Autonomously optimizes a skill via repeated evals, scoring, and prompt mutations. |
| [find-skills](find-skills/SKILL.md) | Helps discover and install agent skills based on what the user wants to do. |
| [skill-creator](skill-creator/SKILL.md) | Creates a new skill end-to-end with test cases and quantitative evals. |
| [skill-optimizer](skill-optimizer/SKILL.md) | Improves an existing skill through a guided, interactive diagnostic loop with checkpoints. |
| [write-a-skill](write-a-skill/SKILL.md) | Quick SKILL.md draft without tests or evals. |
